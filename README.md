# 智慧小苗苗 - 6岁学习伙伴

专为6岁儿童设计的综合性学习应用，包含识字、数学、逻辑思维和英语启蒙。

## 功能特点

- **汉字学习**：学习基础汉字，包括笔画、拼音和组词
- **英语启蒙**：学习英文字母、基础单词和简单句子
- **数学天地**：练习加减法运算和数字认知
- **逻辑思维**：培养观察力和逻辑推理能力
- **进度跟踪**：可视化学习进度和成就系统
- **响应式设计**：适配各种设备屏幕

## 技术实现

- **PWA支持**：可安装为桌面应用，支持离线使用
- **Service Worker**：实现缓存策略，提升加载速度
- **Manifest.json**：应用元数据配置
- **响应式布局**：使用CSS Grid和Flexbox实现

## 使用方法

1. 直接在浏览器中打开 `学习_修复版.html` 文件
2. 或者将项目部署到Web服务器上访问

## 启用GitHub Pages

要通过GitHub Pages访问此应用，请按照以下步骤操作：

1. 访问 [GitHub仓库设置页面](https://github.com/xcc0535/children-learning-app/settings)
2. 在左侧菜单中点击 "Pages"
3. 在 "Source" 部分，选择 "Deploy from a branch"
4. 在 "Branch" 下拉菜单中选择 "main"
5. 在文件夹选择中选择 "/ (root)"
6. 点击 "Save" 保存设置
7. 等待几分钟后，应用将可通过以下链接访问：
   https://xcc0535.github.io/children-learning-app/学习_修复版.html

## 文件说明
