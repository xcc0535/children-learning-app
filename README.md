# 智慧小苗苗 - 6岁学习伙伴

专为6岁儿童设计的综合性学习应用，包含识字、数学、逻辑思维和英语启蒙。

## 功能特点

- 识字学习：包含常用汉字的学习和练习
- 数学启蒙：基础加减法运算练习
- 逻辑思维：图形识别和简单逻辑推理
- 英语启蒙：基础英语单词学习
- 进度跟踪：学习进度记录和可视化展示
- 响应式设计：适配各种设备屏幕

## 技术实现

- HTML5 + CSS3 + JavaScript
- PWA (Progressive Web App) 支持
- 本地存储 (localStorage)
- 语音合成 (Speech Synthesis API)
- 离线访问支持 (Service Worker)

## 如何使用

1. 直接在浏览器中打开应用
2. 可以将应用安装到主屏幕，像原生应用一样使用
3. 应用支持离线访问
